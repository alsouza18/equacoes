🤔 O que é uma Equação do 1° Grau?

Uma equação do 1° grau é uma igualdade matemática que possui uma incógnita (geralmente representada pela letra x) elevada à primeira potência.

ax + b = c

🔍 Onde:

  • 📌 a é o coeficiente de x (a ≠ 0)
  • 📌 b é o termo independente
  • 📌 c é o resultado da equação
  • 📌 x é a incógnita (valor que queremos descobrir)

🎯 Passos para Resolver:

1
Isolar os termos com x
Mova todos os termos que contêm x para um lado da equação
2
Isolar os números
Mova todos os números sem x para o outro lado da equação
3
Dividir pelo coeficiente
Divida ambos os lados pelo coeficiente de x
4
Verificar a resposta
Substitua o valor encontrado na equação original

🚀 O que é uma Equação do 2° Grau?

Uma equação do 2° grau é uma igualdade matemática onde a maior potência da incógnita x é 2 (x²).

ax² + bx + c = 0

🔍 Onde:

  • 📌 a é o coeficiente de x² (a ≠ 0)
  • 📌 b é o coeficiente de x
  • 📌 c é o termo independente
  • 📌 x é a incógnita (pode ter 0, 1 ou 2 soluções)

📐 Fórmula de Bhaskara

x = (-b ± √(b² - 4ac)) / 2a

Δ (Delta) = b² - 4ac

🎯 Passos para Resolver:

1
Identificar os coeficientes
Encontre os valores de a, b e c na equação
2
Calcular o discriminante (Δ)
Δ = b² - 4ac
3
Analisar o Delta
• Se Δ > 0: duas soluções reais diferentes
• Se Δ = 0: uma solução real (raiz dupla)
• Se Δ < 0: não há soluções reais
4
Aplicar a fórmula de Bhaskara
x = (-b ± √Δ) / 2a
